# Ki Tools Documentation

Official documentation for all the Ki Tools of the Canto Basel Stadt.

[![Deploy VitePress site to Pages](https://github.com/DCC-BS/documentation/actions/workflows/deploy.yml/badge.svg)](https://github.com/DCC-BS/documentation/actions/workflows/deploy.yml)

## Tech Stack

- [VitePress](https://vitepress.dev/) - Static site generator
- [Vue 3](https://vuejs.org/) - Progressive JavaScript framework
- [Tailwind CSS](https://tailwindcss.com/) - Utility-first CSS framework
- [Bun](https://bun.sh/) - Fast JavaScript runtime
- [TypeScript](https://www.typescriptlang.org/) - Typed JavaScript
- [Playwright](https://playwright.dev/) - End-to-end testing

## Development

```bash
# Install dependencies
bun install

# Start dev server
bun run dev

# Build for production
bun run build
```
